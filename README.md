# heroesofcode-git

heroesofcode-git is a command-line tool designed to manage and clone repositories from the Heroes of Code organization.

> [!WARNING]
> This tool is intended primarily for **Heroes of Code maintainers**.

## Installation

```sh
cargo install --git https://github.com/heroesofcode/heroesofcode-git
```

## Usage

### List all repositories
```sh
hoc repos
```

### Clone repositories interactively
```sh
hoc clone
```

### Clone all repositories
```sh
hoc all
```

### Show open pull requests
```sh
hoc pr
```

### Help
```sh
hoc --help
```
